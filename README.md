# ComfyUI-Toonflow

A set of ComfyUI nodes inspired by the [Toonflow-app](https://github.com/HBAI-Ltd/Toonflow-app) project, designed to bring powerful AI-assisted storyboard and script generation workflows directly into ComfyUI.

## Features

- **Toonflow Prompt Loader**: Access curated system prompts for various roles (Story Master, Outline Master, Director, Shot Master, Image Prompt Optimizer, etc.).
- **Toonflow Asset Definition**: Define characters, scenes, and props with names and descriptions.
- **Toonflow Asset Group**: Combine multiple assets into a group for easier management.
- **Toonflow LLM Prompt Builder**: Intelligently assembles storyboard descriptions, asset information, style settings, and aspect ratios into a comprehensive prompt ready for your LLM.
- **Toonflow Storyboard Parser**: Parses structured JSON storyboard data from LLM outputs into individual image prompts.
- **Toonflow Script Parser**: Cleans and formats script content generated by LLMs.

## Installation

1.  Navigate to your ComfyUI `custom_nodes` directory.
2.  Clone this repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/ComfyUI-Toonflow.git
    ```
3.  No external dependencies required (standard Python libraries only).

## Workflow Guide

### 1. Asset Management
- Use **Toonflow Asset Definition** nodes to create your characters ("Wang Lin", "Wearing black robe..."), scenes, and props.
- Connect them to a **Toonflow Asset Group** node to bundle them together.

### 2. Prompt Construction
- Connect your Asset Group to the **Toonflow LLM Prompt Builder**.
- Input your raw storyboard description (e.g., "Wang Lin walks by the cliff").
- Select your desired Style and Aspect Ratio.
- The node will output a formatted text block that includes the system instruction, style guides, asset details, and your raw description.

### 3. LLM Processing (Using your own nodes)
- Feed the output from **Toonflow LLM Prompt Builder** into your preferred LLM node (e.g., ChatGPT, Claude, DeepSeek nodes).
- The LLM will use the context provided to generate high-quality, consistent image prompts.

### 4. Parsing & Generation
- If the LLM returns a list of prompts (JSON), use **Toonflow Storyboard Parser** to extract them.
- Send the final prompts to your **Clip Text Encode** and **KSampler** nodes.

## Nodes

- **Toonflow Prompt Loader**: Loads Toonflow's specialized system prompts.
- **Toonflow Asset Definition**: Creates an asset object.
- **Toonflow Asset Group**: Aggregates assets.
- **Toonflow LLM Prompt Builder**: Prepares the context-rich prompt for the LLM.
- **Toonflow Storyboard Parser**: Extracts prompts from JSON output.
- **Toonflow Script Parser**: Cleans script text.
