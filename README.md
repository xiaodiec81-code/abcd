# ComfyUI-Toonflow

A set of ComfyUI nodes inspired by the [Toonflow-app](https://github.com/HBAI-Ltd/Toonflow-app) project, designed to bring powerful AI-assisted storyboard and script generation workflows directly into ComfyUI.

## Features

- **Prompt Loader**: Access curated system prompts for various roles (Story Master, Outline Master, Director, Shot Master, Image Prompt Optimizer, etc.) directly within ComfyUI.
- **Simple LLM**: A lightweight node to interact with OpenAI-compatible LLM APIs (GPT-4, DeepSeek, Claude, etc.) to execute these prompts.
- **Storyboard Parser**: Parses structured JSON storyboard data from LLM outputs into individual image prompts.
- **Script Parser**: Cleans and formats script content generated by LLMs.

## Installation

1.  Navigate to your ComfyUI `custom_nodes` directory.
2.  Clone this repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/ComfyUI-Toonflow.git
    ```
3.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

### 1. Storyline Generation
- Use **Toonflow Prompt Loader** to load the "Story Master (Storyline)" prompt.
- Connect it to **Toonflow Simple LLM** along with your novel text.
- The output will be a structured analysis of your story.

### 2. Outline Generation
- Use **Toonflow Prompt Loader** to load the "Outline Master (Episodes)" prompt.
- Feed the storyline analysis into **Toonflow Simple LLM**.
- The output will be a JSON-formatted list of episode outlines.

### 3. Storyboard Generation
- Use **Toonflow Prompt Loader** to load the "Shot Master (Segment to Prompts)" prompt.
- Feed a specific episode script or outline segment into **Toonflow Simple LLM**.
- Connect the output to **Toonflow Storyboard Parser** to extract clean image prompts.

### 4. Image Generation
- Use the parsed prompts with your favorite **Clip Text Encode** and **KSampler** nodes to generate storyboard images using SDXL, Flux, or other models.

## Nodes

- **Toonflow Prompt Loader**: Selects from a list of pre-defined, optimized system prompts.
- **Toonflow Simple LLM**: Sends prompts to an LLM API. Requires an API Key and Base URL.
- **Toonflow Storyboard Parser**: Extracts prompts from JSON output.
- **Toonflow Script Parser**: Cleans script text (removes markdown code blocks).
